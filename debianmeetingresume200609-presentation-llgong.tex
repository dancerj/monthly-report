 \begin{itemize}[<+->]
  \item もしかしてあまりはやっていない？
  \item カーネルコーディングなど必要な場面は多い
  \item emacs buffer で eval しながらコードがかける言語がうらやましい

  \item ほとんどのプログラム言語には対話インタフェースがあり、
	ためしながらコードが書ける
 \end{itemize}
\end{frame}
