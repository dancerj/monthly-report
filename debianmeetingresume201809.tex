\subsection{第165回東京エリアDebian勉強会}

2018年8月18日(土)に第165回東京エリアDebian勉強会を開催しました。会場は東銀座にある朝日ネットさんをお借りして行いました。参加者は10名でした。

セミナーは、DebConf18に参加した人たちを交えて「台湾で開催したDebConf18に参加して気づいたことや学んだことの雑談会」をBoF形式で行いました。DebConf18に参加できなかった方々からはセミナーやイベントがどのような内容や雰囲気であったかを共有する貴重な機会になりました。

BoFのまとめた内容は、次のURLで見ることができます。

\url{https://gobby.debian.org/export/debconf18/bof/tokyodebian}

利用したツールは、「gobby」というBoFでよく使うツールを使いました。

gobbyの簡単な説明は、「\url{https://wiki.debian.org/gobby.debian.org}」に情報があり、以下のコマンドでインストール、記事の参照また編集ができます。

\begin{commandline}
$ sudo apt-get install gobby
$ gobby infinote://gobby.debian.org/debconf18/bof/tokyodebian
\end{commandline}
