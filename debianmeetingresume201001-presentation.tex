\emtext{BSP についての説明} 

\begin{frame}{BSPとは}
バグを潰しまくるパーティです。バグ増やさないようにしましょう。
\end{frame}

\begin{frame}{RC バグの対応}
\begin{enumerate}
 \item RCバグを直したい人は \#debian-bugs にログインしてください。
 \item \#debian-bugs で 対象のバグを直している人がいないか確認します。
 \item \url{http://wiki.debian.org/BSP2010/CoordinationTokyoMoenchengladbach}
      にRCバグを登録してください。
 \item バグを直します。
 \item その辺にいる Debian Developerに相談して、NMUしてもらいます。 
\end{enumerate}
\end{frame}
